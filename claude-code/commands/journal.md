Show the latest Journal entries from `memory/journal.md`.

## Steps

1. Read `memory/journal.md` (or create it if missing)
2. Display the last 10 entries in a summary table
3. Show statistics: count by entry type (DECISION/FINDING/HYPOTHESIS/ACTION/CORRECTION/QUESTION)

## Output Format

```
## Journal Summary (last 10 entries)

| Time | Type | Summary |
|------|------|---------|
| ... | DECISION | ... |

**Stats**: N decisions, N findings, N hypotheses, N actions, N corrections, N questions
```
